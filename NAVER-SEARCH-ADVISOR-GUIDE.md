# 🔍 네이버 서치어드바이저 등록 가이드

## 개요

네이버 서치어드바이저에 사이트를 등록하여 네이버 검색 결과에 노출되도록 합니다.

## 등록 방법

### 1단계: 네이버 서치어드바이저 접속

1. [네이버 서치어드바이저](https://searchadvisor.naver.com/) 접속
2. 네이버 계정으로 로그인

### 2단계: 사이트 등록

1. **"웹마스터 도구"** 메뉴 클릭
2. **"사이트 추가"** 버튼 클릭
3. 사이트 URL 입력: `https://lanage-convert.vercel.app`
4. 사이트 이름 입력: "나만의 언어 생성기"
5. **"확인"** 버튼 클릭

### 3단계: 사이트 소유권 확인

네이버 서치어드바이저는 사이트 소유권 확인을 위해 다음 방법 중 하나를 선택할 수 있습니다:

#### 방법 1: HTML 메타 태그 (추천)

1. **"HTML 태그"** 방식 선택
2. 제공된 메타 태그 복사
   - 예: `<meta name="naver-site-verification" content="YOUR_VERIFICATION_CODE" />`
3. `app/layout.js` 파일에서 주석 처리된 부분을 찾아 메타 태그 추가:
   ```jsx
   {/* 네이버 서치어드바이저 사이트 소유권 확인 */}
   <meta name="naver-site-verification" content="YOUR_VERIFICATION_CODE" />
   ```
4. 변경사항 커밋 및 푸시
5. 배포 완료 후 네이버 서치어드바이저에서 **"확인"** 버튼 클릭

#### 방법 2: HTML 파일 업로드

1. **"HTML 파일"** 방식 선택
2. 제공된 HTML 파일 다운로드
3. `public` 폴더에 파일 업로드
4. 배포 후 네이버 서치어드바이저에서 확인

### 4단계: 사이트맵 제출

1. 네이버 서치어드바이저 대시보드에서 사이트 선택
2. **"요청"** > **"사이트맵 제출"** 클릭
3. 사이트맵 URL 입력: `https://lanage-convert.vercel.app/sitemap.xml`
4. **"확인"** 버튼 클릭

### 5단계: robots.txt 확인

현재 `public/robots.txt` 파일이 올바르게 설정되어 있습니다:
```
User-agent: *
Allow: /

Sitemap: https://lanage-convert.vercel.app/sitemap.xml
```

## 확인 방법

### 1. 배포 확인

1. https://lanage-convert.vercel.app 접속
2. 우클릭 > "페이지 소스 보기"
3. `<head>` 태그에 메타 태그들이 있는지 확인

### 2. 네이버 서치어드바이저 확인

1. 네이버 서치어드바이저 대시보드 접속
2. 사이트 상태 확인
3. 소유권 확인 완료 여부 확인

### 3. 검색 결과 확인

- 네이버 검색에서 사이트가 노출되는지 확인
- 보통 등록 후 **1-2주** 소요

## 예상 시간

- **사이트 등록**: 즉시
- **소유권 확인**: 몇 분 ~ 몇 시간
- **사이트맵 크롤링**: 1-2일
- **검색 결과 노출**: 1-2주

## 주의사항

1. **메타 태그는 삭제하지 마세요**
   - 소유권 확인 후에도 메타 태그를 유지해야 합니다
   - 삭제하면 소유권 확인이 해제될 수 있습니다

2. **사이트맵 정기 업데이트**
   - 새로운 페이지가 추가되면 사이트맵이 자동으로 업데이트됩니다
   - 네이버 서치어드바이저에서 주기적으로 크롤링합니다

3. **robots.txt 확인**
   - 크롤러가 차단되지 않도록 확인
   - 현재 설정은 정상입니다

## 문제 해결

### 소유권 확인 실패

1. 메타 태그가 올바르게 추가되었는지 확인
2. 배포가 완료되었는지 확인
3. 브라우저 캐시 삭제 후 다시 확인
4. 몇 시간 후 다시 시도

### 검색 결과 미노출

1. 사이트맵이 제출되었는지 확인
2. robots.txt가 크롤러를 차단하지 않는지 확인
3. 콘텐츠가 충분한지 확인
4. 1-2주 대기 (크롤링 시간 필요)
