#### 4️⃣ 키보드 단축키 확장 ✅
**소요 시간**: 25분  
**구현 내용**:
- ✓ `useKeyboardShortcuts.js`: 단축키 Hook
- ✓ `ShortcutsHelpModal.js`: 단축키 안내 모달
- ✓ Mac/Windows 키 자동 감지
- ✓ 10가지 주요 단축키

**전체 단축키 목록**:
| 단축키 | Mac | 기능 |
|--------|-----|------|
| `Ctrl+Z` | `⌘Z` | 실행 취소 (Undo) |
| `Ctrl+Shift+Z` | `⌘⇧Z` | 다시 실행 (Redo) |
| `Ctrl+Enter` | `⌘⏎` | 암호화 |
| `Ctrl+Shift+Enter` | `⌘⇧⏎` | 복호화 |
| `Ctrl+S` | `⌘S` | 프리셋 저장 |
| `Ctrl+K` | `⌘K` | 규칙 추가 |
| `Ctrl+F` | `⌘F` | 규칙 검색 |
| `Ctrl+/` | `⌘/` | 사용 가이드 |
| `Ctrl+B` | `⌘B` | 백업/복원 |
| `Esc` | `Esc` | 모달 닫기 |

**UI 위치**: 네비게이션 바 → ⌨️ 단축키 버튼

**Hook 사용법**:
```javascript
useKeyboardShortcuts({
  onEncode: encode,
  onDecode: decode,
  onSavePreset: () => setShowPresetModal(true),
  onAddRule: addRule,
  onSearch: () => setShowSearch(prev => !prev),
  onHelp: () => setShowQuickGuide(true),
  onBackup: () => setShowBackupModal(true),
});
```
