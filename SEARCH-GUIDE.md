## 🚀 빠르게 노출시키는 방법

### Google Search Console

1. [Google Search Console](https://search.google.com/search-console) 접속
2. 속성 선택: `lanage-convert.vercel.app`
3. **왼쪽 메뉴 → "URL 검사"**
4. URL 입력: `https://lanage-convert.vercel.app`
5. **"색인 생성 요청"** 클릭

### 네이버 서치어드바이저

1. [네이버 서치어드바이저](https://searchadvisor.naver.com/) 접속
2. 사이트 선택
3. **"요청" → "웹 페이지 수집"**
4. URL 제출: `https://lanage-convert.vercel.app`
5. **"요청" → "사이트 간단 체크"** 실행

## 📞 문제 해결

### Q: 2주가 지났는데도 검색이 안 돼요
A: 
1. Google Search Console에서 인덱싱 상태 확인
2. "색인 생성 요청" 다시 클릭
3. sitemap.xml 재제출

### Q: 특정 키워드로 검색이 안 돼요
A: 
1. 경쟁이 심한 키워드는 시간이 더 필요
2. 롱테일 키워드 시도 (예: "무료 언어 생성기")
3. 콘텐츠에 키워드 자연스럽게 포함

### Q: 구글에는 나오는데 네이버에는 안 나와요
A:
1. 네이버는 구글보다 느림 (2-4주 소요)
2. 네이버 서치어드바이저에서 재수집 요청
3. 한국어 콘텐츠 더 추가
